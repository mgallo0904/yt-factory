## For AI Agents

### Working In This Directory
- Markdown files are auto-generated by `generate_scripts.py`; manual edits may be overwritten if the script is re-run
- JSON metadata files are read by `generate_thumbnails.py`, `prep_voiceover.py`, and `download_stock.py`
- Preserve existing content when adding new videos; use the next available index for new entries

### Testing Requirements
- Validate JSON metadata is well-formed before running downstream scripts
- Check that markdown files contain frontmatter or expected sections if consumed by other scripts

### Common Patterns
- Naming: `<NN>_<sanitized_title_snake_case>.<ext>`
- JSON metadata typically contains: title, description, tags, hook_text, target_keywords

## Dependencies

### Internal
- `../generate_scripts.py` — script generator that writes here
- `../generate_thumbnails.py` — reads metadata for thumbnail text
- `../prep_voiceover.py` — reads scripts to produce voiceover text
- `../download_stock.py` — reads metadata for stock footage search queries
